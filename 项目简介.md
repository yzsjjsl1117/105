# 瀹岭 — 高山绿茶·红茶 自营电商平台

## 项目概览

**瀹岭** 是一个单品牌茶叶自营电商平台，源头直供安徽黄山高山绿茶与红茶，面向国内消费者。项目从静态展示站重构为全栈电商，覆盖从浏览、加购、下单到支付、订单管理的完整交易链路。

- **生产地址**：[https://105-nine.vercel.app](https://105-nine.vercel.app)
- **GitHub**：[https://github.com/yzsjjsl1117/105](https://github.com/yzsjjsl1117/105)

## 功能模块

### 前台商城
- **首页** — 品牌展示（Hero 视差、品牌故事轮播、制茶工艺时间线、产品陈列）
- **产品详情** — 左右分栏布局，宣传海报纵向展示产地/外形/香气/汤色/冲泡
- **购物车** — 游客 localStorage + 登录数据库 Hybrid 模式，自动合并
- **结算下单** — 地址选择、库存实时校验（原子 updateMany）、事务创建订单
- **模拟支付** — 微信/支付宝选择 + 30 分钟倒计时 + 订单状态流转
- **用户系统** — 注册/登录/忘记密码/个人中心/地址管理/订单列表

### 管理后台 (`/admin`)
- **Dashboard** — 今日订单、待处理、商品总数、近 7 日销售统计
- **商品管理** — 新建/编辑/删除商品，Supabase Storage 图片上传
- **订单管理** — 状态筛选、发货/完成操作、订单详情查看

## 数据库

11 张核心表：User、Product、Category、CartItem、Order、OrderItem、PaymentRecord、ShippingAddress、PasswordResetToken、RateLimitRecord、VerificationToken

## 订单状态机

```
pending_payment → paid → shipped → completed
                      ↘ expired（30 分钟超时）
                      ↘ cancelled（用户手动取消）
```

## 本地开发

```bash
# 安装依赖
npm install

# 生成 Prisma Client
npx prisma generate

# 启动开发服务器（建议避开默认端口 105，因 Turbopack 编译可能触发系统资源耗尽）
npm run dev -- -p 3106

# 运行种子数据
npm run seed

# 类型检查
npx tsc --noEmit
```

## 环境变量

项目根目录 `.env` 文件需配置：

| 变量 | 说明 |
|------|------|
| `DATABASE_URL` | 本地开发直连 Supabase |
| `DATABASE_POOLER_URL` | 生产环境 Supavisor IPv4 连接池 |
| `AUTH_SECRET` | NextAuth JWT 签名密钥 |
| `SUPABASE_URL` | Supabase 项目地址 |
| `SUPABASE_SERVICE_KEY` | Supabase Storage API 密钥 |

## 已知限制

- **支付**：当前为模拟支付，取得企业支付资质后替换 `POST /api/payment/pay` 内部实现即可
- **搜索/筛选**：产品数量少（3-10 SKU），暂未实现搜索功能
- **邮件服务**：需配置 SMTP 环境变量后密码重置邮件功能才正式生效
